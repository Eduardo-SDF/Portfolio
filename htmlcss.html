Essas foram as tecnologias utilizadas para desenvolver o portfolio visualizado:

HTML (Hypertext Markup Language) é a linguagem de marcação usada para criar a estrutura e o conteúdo de uma página web. Ela consiste em uma série de elementos, como tags e atributos, que são usados para definir a hierarquia e o significado do conteúdo, como títulos, parágrafos, imagens, links e muito mais. O HTML fornece a estrutura básica de uma página web e determina como o conteúdo é organizado e apresentado no navegador.

CSS é uma linguagem de estilo que complementa o HTML, permitindo que os desenvolvedores controlem a aparência e o layout do conteúdo HTML. Com o CSS, é possível definir cores, fontes, espaçamento, posicionamento e outros aspectos visuais de elementos HTML. Ele permite separar o conteúdo da apresentação, facilitando a manutenção e o design responsivo das páginas web.
